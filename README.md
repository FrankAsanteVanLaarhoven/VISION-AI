# QEP-VLA Platform

Quantum-Enhanced Privacy-Preserving Vision-LiDAR-AI Navigation Platform

## Getting Started

To run the application, follow these steps:

1. **Ensure Python 3 is installed.**
2. **Navigate to the project root directory** in your terminal.
3. **Run the startup script:**
   ```bash
   ./start_app.sh
   ```
   Alternatively, you can run the Python script directly:
   ```bash
   python3 start_app.py
   ```

This will start a local web server on `http://localhost:3000` and automatically open your browser to the landing page (`landing-react.html`).

## Important Notes

- The main landing page is located at `http://localhost:3000/landing-react.html`
- This landing page showcases the QEP-VLA platform with interactive 3D elements and advanced UI components
- To access the dashboard, click the "Launch Dashboard" button in the navigation bar
